### 🏗️ Current Project

**[Payments & Ledger Engine](https://github.com/ishwetraj17/payments-and-ledger-engine)** — A production-grade fintech billing and payments platform built to understand how financial correctness is actually engineered at scale.

- Double-entry ledger with DB-trigger-enforced immutability
- 3-layer idempotency (PostgreSQL + Redis NX + Redis response cache)
- Transactional outbox with lease + heartbeat model
- ASC 606 revenue recognition with daily amortization
- Distributed locks with fencing tokens
- 69 Flyway migrations · 2000+ tests · 80+ architecture docs
